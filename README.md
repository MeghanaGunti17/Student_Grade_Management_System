# 🎓 CampusIQ - Student Grade Management System

A full-stack MERN application for managing student academic records, attendance, results, analytics, and performance reports.

## 🚀 Features

### 👨‍💼 Admin

* Manage students and faculty
* View academic analytics
* Monitor department performance
* Grade distribution analysis
* Attendance insights

### 👨‍🏫 Faculty

* Add and manage student marks
* Publish examination results
* Manage attendance records
* Track student performance

### 👨‍🎓 Student

* View academic results
* Check CGPA and pass percentage
* Download PDF reports
* View performance analytics

## 🛠 Tech Stack

### Frontend

* React.js
* React Router
* Axios
* Recharts
* CSS3

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### Authentication

* JWT Authentication
* Role-Based Access Control

### Additional Features

* PDF Report Generation
* Real-Time Updates using Socket.IO
* Responsive Design
* Analytics Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/CampusIQ.git
cd CampusIQ
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
